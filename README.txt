- цветовая палитра заменена на спокойный голубой;
- интерфейс сделан более устойчивым для iPhone.

Как проверить:
1. Заменить файлы в GitHub.
2. Подождать 1-2 минуты.
3. Открыть сайт.
4. На iPhone лучше сначала открыть сайт в Safari напрямую, а не со старой иконки.
5. В режиме Дениса проверить вкладки: Давление, Сон, Еда, Самочувствие.
6. В твоём режиме проверить вкладки: Синхронизация, Обзор, Профиль.

Как настроить синхронизацию:
1. В твоём режиме открыть Синхронизацию.
2. Вставить Supabase URL.
3. Вставить anon public key.
4. Придумать room code, например denis-room.
5. Ввести email.
6. Нажать Сохранить ключи.
7. Нажать Отправить ссылку входа.
8. Открыть письмо и перейти по ссылке.
9. Вернуться в приложение и нажать Проверить вход.
10. Затем нажать Сохранить в облако.
11. На устройстве Дениса ввести те же URL, anon key, room code, но уже его email.
