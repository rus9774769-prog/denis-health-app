1. index.html
2. manifest.webmanifest
3. sw.js

Иконки можно оставить старые.

Как настраивать:
1. Открой сайт
2. Переключись в 'Твой режим'
3. Открой 'Синхронизация'
4. Вставь Supabase URL
5. Вставь anon public key
6. Придумай room code, например denis-room
7. Введи свой email
8. Нажми 'Сохранить ключи'
9. Нажми 'Отправить ссылку входа'
10. Открой письмо и перейди по ссылке
11. Вернись в приложение и нажми 'Проверить вход'
12. Нажми 'Сохранить данные в облако'
